# sports-analytics-innovation
# Project Overview
**The "Power-Surge" Predictor:** This is a predictive analytics tool designed to forecast a player's season-level Home Run (HR) production using early-season counting statistics.

# Decision-Making Problem
This tool supports the **General Manager (GM)** during the Trade Deadline. It helps identify "undervalued" power hitters—players whose current HR totals are low, but whose secondary stats (like Walks and Doubles) suggest a massive power breakout is coming.

# Proposed Analytics Approach
We utilize a **Multiple Linear Regression** model. The approach uses readily available season-level statistics including Walks (BB), Doubles (2B), Triples (3B), Stolen Bases (SB), and Player Age. 

# Use by Decision Makers
A GM or Head of Scouting will interact with a "Target List" generated by this tool. The list ranks potential trade targets by their "Expected HR" vs. "Actual HR," allowing the front office to bid on players before their market value skyrockets.

This currently represents the **Creative Phase**. We have identified a specific organizational friction (overpaying for power) and articulated a data-driven way to solve it using historical performance benchmarks.
